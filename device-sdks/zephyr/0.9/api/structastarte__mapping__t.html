const char* astarte_mapping_t::regex_endpoint

Modified mapping endpoint for regex matching.

The original mapping endpoint can contain parameters in the form %{([a-zA-Z_][a-zA-Z0-9_]*)}. In this modified endpoint all the parameters have been substituted by the regex pattern: [a-zA-Z_]+[a-zA-Z0-9_]* to ease pattern matching.
